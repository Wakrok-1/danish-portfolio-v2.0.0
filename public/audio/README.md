# Audio File

Place your ambient background music file here as:

```
public/audio/relaxed.mp3
```

Requirements:
- Format: MP3
- Recommended volume: low/soft ambient music
- The track loops automatically
- Suggested volume level when encoding: ~60% to avoid being too loud

The audio player component handles:
- Auto-play on first user interaction (browser policy)
- Mute/unmute toggle
- Visual equalizer animation when playing
- Graceful fallback when no file is present

You can use royalty-free ambient music from sources like:
- https://freemusicarchive.org
- https://pixabay.com/music
- https://soundcloud.com (Creative Commons)
